<threat_model>
This plan exposes the public API. Main risks are API footguns, mutable internal state leaks, and renderer coupling.

- Threat: host app mutates internal state through snapshots. Mitigation: return cloned/frozen readonly snapshots from `getState()`, `getElements()`, and feature queries.
- Threat: unsupported methods silently do nothing. Mitigation: throw typed `UnsupportedOperationError` for later-phase operations.
- Threat: core imports renderer types. Mitigation: adapter contract uses map-neutral `Coordinate` and event shapes only.
</threat_model>
