<threat_model>
This plan creates package/config files only. Main risks are supply-chain exposure and accidental publishability.

- Threat: workspace packages are accidentally publishable before API stabilization. Mitigation: every package manifest created in this plan must contain `"private": true`.
- Threat: adapter packages accidentally install renderer dependencies into core. Mitigation: Phase 1 adapter packages are placeholders and core package has no Leaflet/MapLibre dependency.
- Threat: scripts silently skip packages. Mitigation: root scripts use pnpm recursive/project commands and TypeScript references include every package/example folder created in this plan.
</threat_model>
