<threat_model>
This plan mutates existing geometry during drawing. Main threats are accidental snapping, corrupt way sequences, stale visuals, and dependency leakage.

- Threat: user unintentionally creates shared topology. Mitigation: snapping is disabled by default and must be enabled through host API/options.
- Threat: edge snapping corrupts an existing closed way. Mitigation: use `PrimitiveStore.updateWayNodes` closure rules and tests that assert both old and new way node arrays.
- Threat: core imports renderer libraries. Mitigation: core uses `RendererAdapter.project` only; acceptance checks grep for Leaflet imports in core.
- Threat: snap dependency becomes public API. Mitigation: expose SDK-owned snap types only, not RBush/Turf types.
</threat_model>
