<threat_model>
This plan changes shared primitive mutation semantics. Main threats are accidental neighbor edits, broken references after delete/detach, stale adapter state, and import data loss.

- Threat: moving a shared node silently changes adjacent geometry. Mitigation: this is required by D-05 and must emit affected feature/primitive events for host awareness.
- Threat: detach duplicates tags/timestamps incorrectly. Mitigation: clone coordinates into new nodes with current timestamps and keep way/feature tags unchanged.
- Threat: delete removes shared nodes. Mitigation: reference checks and tests assert shared nodes survive.
- Threat: import rebuild loses unknown valid tags. Mitigation: shared fixture tests assert unknown tags survive export.
</threat_model>
