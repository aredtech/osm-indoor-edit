<threat_model>
This plan exposes relation mutation APIs. Main threats are broken member references, accidental recursive relation corruption, and renderer attempts to draw non-geometric relation features.

- Threat: appending broken members creates invalid export. Mitigation: primitive-store relation member mutations validate references before commit.
- Threat: relation-backed features trigger Leaflet rendering errors. Mitigation: editor does not call adapter commit/update for `geometryType: "relation"` records.
- Threat: deleting relation members by ambiguous identity removes the wrong member. Mitigation: remove API must use exact `type/ref/role` or explicit index, with tests.
</threat_model>
