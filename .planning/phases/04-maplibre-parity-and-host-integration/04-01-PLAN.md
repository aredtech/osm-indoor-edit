<threat_model>
This plan introduces a second renderer package. Main threats are renderer leakage into core, source/layer ID collisions with host styles, stale source data, and incomplete cleanup.

- Threat: `maplibre-gl` leaks into core. Mitigation: add MapLibre imports only under `packages/maplibre`; acceptance checks grep `packages/core/src` for `maplibre`.
- Threat: SDK source/layer IDs collide with host map styles. Mitigation: default prefix `osminedit`, configurable `sourcePrefix`, and tests for custom prefix IDs.
- Threat: detach leaves sources, layers, or listeners on the host map. Mitigation: track all source IDs, layer IDs, and listeners; tests assert removal.
- Threat: relation features render as broken geometry. Mitigation: conversion function returns no GeoJSON feature for `geometryType: "relation"`.
- Threat: stale level-filtered source data after `setLevel`. Mitigation: keep committed features in adapter memory and regenerate committed/selection data on level changes.
</threat_model>
