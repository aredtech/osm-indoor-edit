<threat_model>
This plan introduces a concrete renderer dependency. Main threats are renderer leakage into core, event listener leaks, and host-visible layer pollution.

- Threat: Leaflet imports leak into `packages/core`. Mitigation: only `packages/leaflet` imports `leaflet`; add grep acceptance check.
- Threat: adapter detach leaves map listeners/layers behind. Mitigation: adapter tracks unsubscribe handlers and clears root groups in tests.
- Threat: host must manage SDK layers. Mitigation: adapter creates an SDK-owned pane/root group and internal subgroups.
</threat_model>
