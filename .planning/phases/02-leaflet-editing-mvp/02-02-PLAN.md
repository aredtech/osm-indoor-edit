<threat_model>
This plan mutates editor primitives from map input. Main threats are invalid exports, partial commits, and drafts leaking into committed state.

- Threat: incomplete polygon finish corrupts primitives. Mitigation: typed error before primitive creation, tests assert no committed elements.
- Threat: missing level silently creates bad indoor data. Mitigation: `startDraw` requires current level.
- Threat: cancelling leaves draft visuals or state. Mitigation: adapter `clearTemporaryFeature`, editor state reset, `drawingCancelled` event test.
</threat_model>
