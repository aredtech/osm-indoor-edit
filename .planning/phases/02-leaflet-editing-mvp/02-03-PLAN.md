<threat_model>
This plan introduces destructive and live edit operations. Main threats are broken way references, invalid polygons, accidental deletion, and stale Leaflet visuals.

- Threat: vertex deletion creates invalid polygons. Mitigation: core validates minimum remaining geometry before mutating.
- Threat: feature delete leaves dangling primitive references. Mitigation: delete feature and related primitives through coordinated store methods; tests assert export remains valid.
- Threat: live drag updates only visuals, not primitives. Mitigation: core tests assert node coordinates and export change after drag events.
</threat_model>
